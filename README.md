# Cite

Agentic RAG with clickable citations and a synced document viewer.

Teams upload their documents and chat with them in natural language. Every answer is grounded by inline citations — clicking one opens the source document at the exact region that produced the answer.

## Status

Under active development.

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind v4 + shadcn/ui (Radix Nova preset)
- Yarn 4

Full stack and deployment instructions will land in this README as the project comes online.
